Save Leads to SalesForce using your Formidable Forms

== Description ==

Add an action to a Formidable Form to save your leads to SalesForce using the Salesforce Web-To-Lead form. No API required.

== Installation ==

1. Upload 'formidable-salesforce-webtolead' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In Formidable, add a Salesforce Web-To-Lead action via the Forms > Settings > Form Actions interface
